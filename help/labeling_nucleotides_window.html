What can you do in this window?

This window appears when a spin labeling site scan is initiated from the EPR/Site scan menu item after the OK button was clicked in the site scan window and the objects to be labeled are nucleotide residues. In a site scan of several chains, this window appears once for every chain (as individual proteins in a complex might be differently labeled).

You can select the type of label and the target temperature for rotamer computations. If you have unlocked functions for experienced users (unlock command or Lock/Unlock (toggle button)), you can explicitly select a rotamer library by file name.

Label type

Here you can select the type of label. Currently, only iodoacetamido-PROXYL attached to 4-thiouracil is supported.

Temperature selection

Currently, two main temperature selections are supported. Use cryogenic for low-temperature experiments on both soluble and membrane proteins. The target temperature of 175 K is close to the glass transition temperature of the solvation shell of soluble proteins as well as the glass transition temperature of lipid bilayers. This was previously the recommended setting for DEER and P-31 Mims ENDOR. However, tests on proteins with known structure indicated that ambient temperature libraries perform slightly better even for experiments performed at cryogenic temperatures.

We now recommend to use ambient for all experiments. This sets the target temperature to 298 K.

It is possible to directly input the target temperature, which leads to Boltzmann scaling of internal rotamer populations. There is no strict theoretical justification for this. It is a reasonable approximation at temperatures close to the ones where rotamer libraries were derived (175 and 298 K), however, the temperature dependence may then be insignificant with respect to uncertainty of the approach anyway. We generally recommend to stick to the preset values for cryogenic and ambient temperature experiments. The direct input feature is provided for tests.

Library selection

This feature is available only if functions for experienced users have been unlocked. A popupmenu lists all rotamer library files that match the previously selected labeling pattern (all .mat files in subdirectory rotamer_libraries). You can directly select one of these libraries. In addition, you need to mark the checkbox use special library:. This choice overrides temperature and label selection.

Currently available rotamer libraries are of the form R1A_T_type.mat, where T is either 175K or 298K and type is 090619 (original MMM libraries for MTSL), culled (all rotamers with respect to chi1 and chi2 removed that were never observed in crystal structures), xray (chi1, chi2 probabilities as in the set of all reliable x-ray structures), flat (equal probabilities for all rotamers), sezer12 (chi1, chi2 probabilities as in the MD computations of D. Sezer, J. H. Freed, B. Roux 2008 J. Phys. Chem. B 122:5755-5767, Table 10), or warsh (chi1, chi2 probabilities as in D. T. Warshaviak, L. Serbulea, K. N. Houk, W. L. Hubbell 2011 J. Phys. Chem. B 115:397-405).
